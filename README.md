# Personal Diary Manager (C++)

A simple console-based diary application built using C++ and file handling.  
This project allows users to store personal diary entries with timestamps and search them by date.

## Features
- Add new diary entries
- View all saved entries
- Search entries by date
- Stores data permanently using files

## Technologies Used
- C++
- File Handling (ifstream, ofstream)
- Object-Oriented Programming
- ctime for timestamps

## How to Run
1. Clone the repository
2. Compile using:
   g++ diary.cpp -o diary
3. Run:
   ./diary

## Learning Outcomes
- Learned file handling in C++
- Practiced OOP concepts
- Understood how to work with time and strings
